Samples are the executable companion to the documentation. {% .lead %}

---

| Sample | Demonstrates |
| --- | --- |
| `samples/Demos` | Roadmap-driven executable demo suite, shared RabbitMQ setup, case catalog |
| `samples/Standalone` | Generic Host, HTTP endpoints, RabbitMQ consumers |
| `samples/Aspire` | Aspire AppHost, API, Worker, Web, RabbitMQ resource |
| `samples/Attention.Primitives` | Attention pattern composed from generic primitives |

`samples/Demos` is intentionally staged. The current phase provides the shared host, broker compose file, case catalog, and README files for the 17 planned demos. Case runners are added incrementally according to `spec/demo-cases-roadmap.md`.
